# 塔里木刷题王

塔里木刷题王是一款面向油气田相关岗位取证、技能竞赛和日常考试复习的题库学习软件。软件产生的初衷，是让甲方、乙方员工都能用同一套完整题库开展备考，减少不同单位、不同管理区之间的信息差，让考试准备回到学习本身，形成更公平的竞争环境。

软件内置全量完整题库，围绕“快速开始、持续复习、错题强化、考前模拟”设计学习流程。它不是简单的题目展示工具，而是把题库、练习、复习、统计、错题和模拟考试整合到一个本地应用里，方便员工在碎片时间和集中备考阶段持续使用。

## 适用场景

- 员工取证考试、岗位等级考试、技能竞赛备考。
- 甲方、乙方人员使用统一题库进行公平复习。
- 管理区、班站、项目部组织集中学习和考前训练。
- 个人长期刷题、错题复盘、每日复习和模拟考试。

## 核心特性

- 全量题库内置：预置天然气净化、轻烃操作、采油、集输、安全风险等相关题库。
- 多题型支持：支持判断题、单选题、多选题等常见考试题型。
- 科学复习节奏：每日复习会根据已学习题目、错误情况、连续答对次数和到期时间生成复习队列。
- 全题库每日复习：不需要在多个题库之间反复切换，可以从全量题库中自动筛选今日应复习内容。
- 错题优先：系统会记录答题结果，优先拉回错得多、掌握不稳定、逾期较久的题目。
- 重难题管理：低正确率题目会进入重点关注范围，方便集中突破。
- 斩题机制：已经稳定掌握的题目可以斩掉，减少重复消耗。
- 模拟考试：支持按题型配置考试题量，进行接近实战的考前训练。
- 学习统计：记录每日答题量、正确率、熟练度和复习进度，便于掌握备考状态。
- 本地运行：学习记录保存在本机，打开即用，适合现场和离线环境。

## macOS 版本

macOS _arm64版本是当前项目的主要维护版本，适合作者日常开发、验证题库、检查功能和快速生成用户发布包。

macOS 版本相关源码位于：

```text
macos/
scripts/build-mac-app.sh
```

构建命令：

```bash
npm run build
bash scripts/build-mac-app.sh
```

生成的安装包位于：

```text
release/
```

## 多平台版本

项目当前规划和维护三个用户版本：

- macOS arm64：面向 Apple Silicon Mac 用户，发布 `.dmg` 安装包。
- Windows：面向 Windows 用户，发布`.exe` 安装版包
- Android：面向手机刷题场景，发布 `.apk` 安装包。

各平台构建命令：

```bash
npm run desktop:pack
npm run android:pack
npm run desktop:pack:windows
```

说明：Windows 版本需要在 Windows 构建环境中执行；Android 版本需要本机配置 JDK、Android SDK 和签名环境。

## 题库与学习方式

软件将题库按岗位和等级组织，用户可以选择单个题库训练，也可以使用全题库复习模式。系统会持续记录每道题的学习情况，包括已答次数、正确次数、错误次数、连续答对次数、最后答题时间和最后错题时间。

每日复习不是固定顺序刷题，而是按学习状态动态生成。系统会优先安排：

- 已经学过但到期需要复习的题。
- 错误次数较多的题。
- 正确率较低的题。
- 连续答对次数较少的题。
- 逾期时间较长的题。

这种方式可以减少“只刷会的题”的无效重复，把时间更多分配给真正影响成绩的薄弱点。
